# SocialMediaTradwives

Visual Settings in Tradwife Related Instagram Content: A Computational Cross National Analysis of the United States and Germany

## Project Overview

This repository contains the notebooks and code used for the analysis conducted in the study "Visual Settings in Tradwife Related Instagram Content: A Computational Cross National Analysis of the United States and Germany".

The project examines how spatial settings are visually represented in Instagram posts associated with the tradwife movement. Using computational methods and manual annotation, the analysis identifies recurring visual environments and compares patterns between accounts based in the United States and Germany.

The study addresses the following research questions:

**RQ1:** Which types of settings are most frequently depicted in Instagram posts by tradwife-associated accounts?

**RQ2:** How do the depicted settings differ between tradwife-associated posts from the United States and Germany?

## Repository Contents

This repository includes notebooks used for data preparation, analysis and visualization. Large files such as images, extracted frames and intermediate datasets are not stored directly in this repository because of GitHub storage limitations.
